About Rese

Rese (/r…õs/, Reliverse) is a blazing fast, minimalist framework designed for modern web, desktop, and mobile applications. Built for Bun, Deno, and Node.js, it delivers a developer-first experience with expressive routing, built-in WebSocket support, and a flexible plugin system.

Inspired by Express, Koa, Hono, Next.js, TanStack Start, and others, Rese leverages native Bun.serve and a cutting-edge radix tree for routing to provide exceptional performance and scalability.

What is Rese?

Rese is an extendible, ultra-fast, minimalist framework for TypeScript, Node.js, Bun, desktop, mobile, and web applications ‚Äî offering a batteries-included developer experience with expressive routing and clean, context-driven middleware.

Examples

Hello, Rese


// index.ts
import { Rese } from "@reliverse/rese";

const app = new Rese();

app.get("/", (ctx) => ctx.sendText("Hello from Rese üåå"));

app.listen({ port: 3000 });

Dynamic Routes


app.get("/user/:username", async (ctx) => {
  const { username } = ctx.params;
  return ctx.sendJson({ profile: username });
});

WebSocket Example


app.ws("/chat", {
  open: (ws) => ws.send("Welcome to Rese chat!"),
  message: (ws, msg) => ws.publish("chat", msg),
});

Plugin Architecture

Rese is built around a lightweight, opt-in plugin system that lets you customize and extend your app without sacrificing performance. Whether you need SSR with React, AI-enabled endpoints, file-based routing, or even full RPC support, there‚Äôs a plugin for that.
